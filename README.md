# 🎬 Netflix Data Analysis Using Python

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Movies and TV Shows dataset using **Python** to understand patterns in Netflix’s global content catalog.

The analysis focuses on identifying trends in **content distribution, release patterns, genres, and production countries**. The project demonstrates how raw data can be cleaned, processed, and transformed into meaningful insights using Python data analysis and visualization techniques.

---

# 📌 Project Overview

The objective of this project is to analyze Netflix’s content dataset and extract valuable insights about movies and TV shows available on the platform.

Using Python and popular data analysis libraries, the dataset is explored through **data cleaning, transformation, and visualization** techniques to understand how Netflix content is distributed across different genres, years, and countries.

A detailed analytical report summarizing the findings is included in this repository.

---

# 📊 Dataset

The dataset contains information about **Netflix Movies and TV Shows available on the platform.**

### Key Attributes

* **show_id** – Unique identifier for each title
* **title** – Name of the movie or TV show
* **type** – Movie or TV Show
* **director** – Director of the content
* **cast** – Main cast members
* **country** – Country of production
* **date_added** – Date when the content was added to Netflix
* **release_year** – Original release year
* **rating** – Content rating classification
* **duration** – Duration of the title
* **listed_in** – Genre categories
